# Contributing

MailInlay is an owner-maintained, all-rights-reserved project.

The sole author and maintainer is Sebastian Pawelczyk
([@SebastianPRM](https://github.com/SebastianPRM)). External pull requests are
not accepted unless the owner has requested the contribution in advance.

Bug reports and suggestions may be submitted as GitHub issues. Submission does
not grant write access, authorship, ownership or a license to the repository.
Whether a proposal is used remains entirely at the owner's discretion.

Security issues must follow [SECURITY.md](SECURITY.md) and must not be posted in
public issues.
